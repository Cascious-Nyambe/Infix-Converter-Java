# Infix to Postfix & Prefix Converter

This Java application converts mathematical expressions from Infix notation (e.g., A + B * C) into Postfix (e.g., ABC*+) and Prefix (e.g., +A*BC) notations using Stack data structures.

This project was developed for the ICT202 Data Structures II course.

## Features
- Converts Infix expressions to Postfix.
- Converts Infix expressions to Prefix.
- Handles operator precedence (^, *, /, +, -).
- Supports parentheses ().

## How to Run

1. Make sure you have Java installed on your computer.
2. Open Terminal or Command Prompt.
3. Navigate to the project folder:
   cd InfixConverter
4. Compile the code:
   javac InfixConverter.java
5. Run the program:
   java InfixConverter
6. Enter an infix expression when prompted (e.g., (A+B)*C).
